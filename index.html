Needs an HTTP server
This demo downloads ~22 MB of ML model weights from Hugging Face. Chrome blocks those network requests when the page is opened directly from a file — it must be served over HTTP.

Option 1 — one command (Python already on your machine):
cd ~/Downloads && python -m http.server 8080 click to copy
Then open http://localhost:8080/docqa-ondevice.html
— or —
Option 2 — drag the file to Netlify Drop (10 seconds, free):
Open app.netlify.com/drop →
